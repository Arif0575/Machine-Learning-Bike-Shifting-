# AI Bicycle Gear Prediction System

Machine-learning-assisted bicycle gear selection system developed as part of a Bachelor of Engineering (Honours) project in Robotics & AI.

This project explored how ride telemetry and sensor data could be used to predict optimal bicycle gear selection using machine learning techniques and embedded hardware.

## Project Overview

The system was designed to collect real-world ride telemetry data and use a machine learning model to predict suitable gear changes during cycling.

The project combined:
- Embedded systems
- Sensor integration
- Data collection and preprocessing
- Machine learning model training
- Hardware/software integration
- Real-world deployment concepts

## System Components

- Raspberry Pi
- Arduino Nano
- Ride telemetry sensors
- Python-based ML pipeline
- Custom 3D-printed mounting solutions
- Mobile interface concepts

## Machine Learning Workflow

1. Ride telemetry data collection
2. Data cleaning and preprocessing
3. Feature extraction
4. MLP (Multi-Layer Perceptron) model training
5. Gear prediction evaluation
6. Embedded deployment experimentation
